# 📷 Camera E-Commerce Website

A multi-page **camera shopping platform** where users can explore camera brands, browse products, view deals, and manage their accounts.
The project focuses on building a **realistic e-commerce user interface with authentication pages, product browsing, and purchase tracking.**

---

# 🚀 Features

## 🛍️ Product & Shopping Features

* Browse cameras from different brands
* Explore product categories and accessories
* Dedicated pages for **camera sales and deals**
* Camera rental section
* Camera gear shopping section
* Product guides for users

---

## 🔐 Authentication System

* **User Registration Page**
* **User Login Page**
* Secure login validation using JavaScript
* Separate UI pages for authentication

Pages involved:

* `login.html`
* `register.html`
* `acc.html`

---

## 📦 Purchase Management

Users can view their purchase details through:

* `purchase-history.html`
* `purchase-history.js`

Features:

* Displays previously purchased items
* Tracks user purchase activity

---
